# 入境中国图文攻略 — 方案 v4

## 设计系统

- **布局**：PPT 式全屏 snap 滚动（`100dvh` × `scroll-snap`），单 HTML 文件零依赖
- **时间线**：左侧固定竖线贯穿全屏，每屏节点+标题在时间线旁，dot 精确居中于竖线
- **内容**：左对齐（非居中），标题在时间线节点右侧，内容卡片在下；三档垂直位置（top/mid/bot）
- **导航**：右侧可拖动横条指示器（参考手机相册拖动体验），带 tooltip 显示序号和标题
- **封面屏**：锁定滚动，选择身份后才解锁并自动跳转
- **配色**：v1 浅色主题 — 暖米白背景 (`#faf9f6`) + 白色实体卡片 + 中国红强调色 (`#c41e3a`)，无 dark mode
- **卡片**：实体白底 + 细微阴影 + 左侧彩色边框变体（红/绿/琥珀/蓝）
- **双语**：CSS `[lang]` 属性切换中/英，语言偏好存入 localStorage
- **响应式**：手机上下排版，中/大屏 (≥1024px) 左右两栏（标题 | 内容卡片）

## 核心决策：短期 / 长期，完全两个版本

用户在首屏选择身份，进入各自专属的完整流程。

| | 🛫 短期游客版 | 🏠 长期居留版 |
|---|---|---|
| **目标用户** | 境外来华，停留 2-30 天 | **已在华**，打算长期定居（>180天） |
| **用户画像** | 韩国周末游、东南亚美食游、欧洲深度游、商务出差 | 签证转居留、工作许可、外籍员工、留学生转定居 |
| **核心痛点** | 支付壁垒、语言障碍、找不到路、上不了 Google | 居留许可、租房、银行开户、医保、子女教育 |
| **入境手续** | 完整保留（签证→通关→登记） | **大幅简化**（用户已入境，跳过签证申请和通关） |
| **信息深度** | 快速上手 | 深入安居指南 |
| **色调** | 暖色（旅行感） | 冷色（专业感） |

### 核心洞察

1. 免签是绝对杠杆（73%免签入境）
2. 韩国+东南亚是最大客源（非欧美）
3. 支付是第一大痛点
4. 自由行>70%，攻略需求远超跟团时代
5. 目的地从北上广深扩展到100+城市
6. 短期要"快速上手"，长期要"安居指南"——这是两版分流的根源
7. **长期版用户已在华**，不需要签证申请和通关教程，需要的是身份转换和安居
