# 入境中国图文攻略

单 HTML 文件，零依赖。PPT 式全屏 snap 滚动，6 语言支持。

## 项目结构

```
HackYourWaytoChina/
├── src/                    # 源代码（直接serve）
│   ├── index.html         # 主页面
│   ├── css/style.css      # 样式
│   ├── js/                # 脚本
│   │   ├── app.js         # 应用逻辑
│   │   ├── data.js        # 数据定义
│   │   ├── detail.js      # 详情页渲染
│   │   └── i18n.js        # 国际化引擎
│   └── i18n/              # 6语言翻译文件
│       ├── zh.json        # 中文
│       ├── en.json        # 英文
│       ├── ja.json        # 日语

│       └── vi.json        # 越南语
├── scripts/               # 工具脚本
│   └── docx2md.py         # 文档转换
└── docs/                  # 文档
    ├── research.md        # 市场调研

    ├── feedback.md        # 反馈记录

```

## 多语言支持

**6种语言完整支持**：中文、英文、日语、韩语、俄语、越南语
- 所有timeline标题、卡片内容、详情页、按钮完整翻译
- 动态语言切换，偏好存入localStorage
- i18n引擎自动加载JSON翻译文件

## 设计系统

- **布局**：`100dvh` × `scroll-snap`，每屏强制对齐，封面屏锁定滚动
- **时间线**：左侧固定竖线贯穿全屏，dot **水平**居中于竖线；内容垂直位置三档可选（`top` / `mid` / `bot`）
- **内容结构**：左对齐，时间线节点 + 标题在上，内容卡片在下；≥1024px 左右两栏（标题侧 | 卡片侧）
- **导航**：右侧可拖动横条指示器，hover 预览标题，拖拽松手 snap 到对应屏
- **配色**：v1 浅色主题 — 暖米白背景 (`#faf9f6`) + 白色实体卡片 + 中国红强调色 (`#c41e3a`)，无 dark mode
- **卡片**：实体白底 + 细微阴影 + 左侧彩色边框变体（红/绿/琥珀/蓝）
- **双语**：CSS `[lang]` 属性切换中/英，偏好存入 localStorage
- **响应式**：手机单列，≥1024px 双栏

## 核心决策：短期 / 长期，完全两个版本

用户在首屏选择身份，进入各自专属流程。

| | 🛫 短期游客版 | 🏠 长期居留版 |
|---|---|---|
| **目标用户** | 境外来华，停留 2-30 天 | **已在华**，打算长期定居（>180天） |
| **用户画像** | 韩国周末游、东南亚美食游、欧洲深度游、商务出差 | 签证转居留、工作许可、外籍员工、留学生转定居 |
| **核心痛点** | 支付壁垒、语言障碍、上不了 Google | 居留许可、租房、银行开户、医保、子女教育 |
| **入境手续** | 完整保留（签证→通关→登记） | **大幅简化**（用户已入境，跳过签证和通关） |
| **信息深度** | 快速上手 | 深入安居指南 |

## 已知缺口

以下问题已通过 [resources.md](resources.md) 调研确认，待下期实现：

| # | 缺口 | 影响 | 计划 |
|---|------|------|------|
| 1 | **语言仅中英** | TOP10 客源国中仅覆盖 3 国（中+英+新加坡），韩国(#1)、俄罗斯(#2)、日本(#8)、越南(#4) 等均缺 | [roadmap.md](roadmap.md) 语言扩展 |
| 2 | **牙科内容薄弱** | S7 仅几行文字，但韩日客群核心诉求之一即牙科；需深入市场调研 | 公司一阶段重点 |
| 3 | **签证缺详情** | S1 自测器仅判断类型，不展示签证要求、材料、申请链接、费用 | [roadmap.md](roadmap.md) 签证详情页 |
| 4 | **医疗旅游缺位** | 2025年128万国际患者(+73.6%)，但仅 L5 几行文字；缺牙科/眼科/中医专题 | [roadmap.md](roadmap.md) 医疗旅游专题 |
| 5 | **短/长版共用配色** | CSS 定义了 `--short-accent` / `--long-accent` 但两版实际使用同一调色板 | 暂缓 |
| 6 | **签证政策俄语区特殊** | 俄罗斯免签至 2026.9.14（非 12.31），自测器未做区分提示 | 待修复 |
