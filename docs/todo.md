## 内容修改工作流

### 如何添加/修改内容

1. **走用户旅程**：不要一股脑填充 research，先自主思考：一个外国人如何一步步完成这个任务？跟着流程写内容
2. **先写中文**（`zh.json`），再翻译到其他 5 语言
3. **i18n 键命名**：使用 `slide.key.subkey` 层级，不要用扁平化点分隔键
4. **fallback**：翻译缺失时自动 fallback 到 en.json

### 如何添加图片

- 图片放在 `src/images/`
- HTML 中使用 `<img>` 标签，通过 `loading="lazy"` 懒加载
- 考虑首屏性能，建议从 S5（绑卡截屏）或 S3（通关流程图）开始试验
